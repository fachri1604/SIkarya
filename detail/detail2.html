Deskripsi Proyek

oleh: Fitri Astuti

Aplikasi berbasis website ini di rancang dengan menggunakan perangkat lunak Sublime Text 3, Xampp, framework Laravel, CorelDRAW X8, Balsamic Mockup sebagai perancangan interface website serta web browser Mozilla Firefox. Metode pengembangan sistem yang digunakan adalah waterfall serta metode pengumpulan data berupa observasi, studi literatur, wawancara dan dokumentasi .Tujuan dari aplikasi ini adalah untuk menyajikan dan menyebarkan informasi mengenai profile sekolah, berita/pengumuman penting sekolah, prestasi yang di dapat oleh sekolah, data staff guru, serta fitur akses materi via website. Serta aplikasi website ini dapat digunakan oleh guru dan siswa dengan otoritas tertinggi yang di pegang oleh admin. Seorang admin dapat menambah, memperbarui dan menghapus berita, prestasi, data guru, serta file materi pelajaran.
